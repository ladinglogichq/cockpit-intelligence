# Cockpit — frontend (Vite + React)

Public marketing site: landing, blog, pricing, explore-data, methodology.

## Quick start

```bash
cp .env.example .env
npm install
npm run dev
```

Dev server: [http://localhost:5173](http://localhost:5173)

## Environment

See `.env.example` — mainly `VITE_SITE_URL` and optional Supabase-related vars.

## Scripts

| Command | Description |
|--------|-------------|
| `npm run dev` | Dev server with HMR |
| `npm run build` | Production build to `dist/` |
| `npm run preview` | Preview production build locally |
